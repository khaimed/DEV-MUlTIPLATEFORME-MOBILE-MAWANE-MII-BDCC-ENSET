# Mobile — Travaux Pratiques ENSET

Ce dépôt regroupe les travaux pratiques réalisés dans le cadre du module de développement mobile.

## Projets

### [ChatAI](ChatAI_Cross/README.md)

Application de chat multiplateforme (Android, iOS, web, Windows, macOS, Linux) type "ChatGPT", développée en Flutter et propulsée par l'API gratuite de Groq (modèles Llama).

- Dossier: `ChatAI_Cross/`
- Stack: Flutter / Dart
- Voir le [README du projet](ChatAI_Cross/README.md) pour la mise en route.

### [IMC App](IMC%20App/README.md)

Application Android native (Java) de calcul de l'IMC (indice de masse corporelle), avec affichage de la catégorie correspondante.

- Dossier: `IMC App/`
- Stack: Android natif / Java
- Voir le [README du projet](IMC%20App/README.md) pour la structure et les fonctionnalités.

## Organisation

```
Mobile/
```
